1. Executive Summary

Project Managers across the organisation currently manage project financials in fragmented, inconsistent ways — relying on manual SAP exports, personal spreadsheets, and ad-hoc forecasting methods. Monthly reporting is a time-consuming burden with no standard format, resulting in inconsistent quality, late detection of cost overruns, and significant PMO overhead.



The Project Financial Management System (PFMS) is a dedicated internal web application that standardises EAC tracking, automates labour cost forecasting from headcount inputs, and uses AI to generate clear, professional project updates — removing the writing burden from Project Managers entirely.



Over time, the platform evolves into a Project Intelligence System — proactively surfacing risks, detecting anomalies, and prompting PMs before problems escalate.

5. Project Intelligence Layer
